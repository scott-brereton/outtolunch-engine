# Sales / RevOps starter pack

Pipeline-focused starter for outtolunch-engine. Six sections covering recent closes, churn, at-risk accounts, top deals in flight, quota attainment, and corrections. Confidence scoring disabled — this pack assumes you're pulling from the CRM as the system of record.

To use:
- cp -R packs/sales-revops/. pack/
- Replace example data with real pipeline data.
- Run npm run validate, then npm run deploy.

See SETUP.md for the LLM-led configuration walkthrough.
