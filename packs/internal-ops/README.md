# Internal Ops starter pack

The hero pack for outtolunch-engine. Eight sections covering leadership, vacation, hires/departures, priorities, incidents, internal tool versions, corrections, and upcoming events. Confidence scoring disabled — this pack assumes you're pulling from systems of record.

To use:
- cp -R packs/internal-ops/. pack/
- Replace example data with real org data.
- Run npm run validate, then npm run deploy.

See SETUP.md for the LLM-led configuration walkthrough.
